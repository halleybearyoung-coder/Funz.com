Funz.com: Read, Draw, and Play Games

Welcome to Funz.com! This is a simple, modern Single Page Application (SPA) designed to provide a fun and engaging environment for users of all ages. It combines interactive reading, a drawing pad, and simple, challenging games, such as our unbeatable Tic-Tac-Toe bot.

The entire application is built using pure HTML, Tailwind CSS, and vanilla JavaScript, making it lightweight, fast, and easy to deploy anywhere.

Unbeatable AI: The Maestro Magnus bot (Player 'O') utilizes a fast heuristic to always block wins and force a draw or victory.

Local Multiplayer: Play against a friend (Human vs. Human).

Responsive Design: Fully optimized for both desktop and mobile devices.
